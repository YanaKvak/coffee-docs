# Обзор приложения "КофеМан"

Приложение позволяет заказывать кофе навынос и управлять бонусным счетом.

## Возможности

* **Заказ заранее** - избегайте очередей
* **История заказов** - повторяйте любимые позиции
* **Персональные скидки** - получайте специальные предложения
